# Последний коммит: Хроники латте-джунглей

Тактический survival в опустевшем опенспейсе.  
Вы — последний выживший разработчик. Пробивайтесь через этажи заброшенного технопарка, сражайтесь с багами, собирайте команду и доберитесь до легендарного сервера, чтобы сделать `git push --force` и перезагрузить реальность.

## Требования

- Rust 1.77 или новее (установите через [rustup](https://rustup.rs/))
- Зависимости проекта автоматически скачиваются Cargo

## Сборка и запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/last-commit/last-commit.git
cd last-commit

# Запуск в режиме разработки
cargo run

# Сборка оптимизированной версии (без отладки)
cargo build --release
# исполняемый файл будет в target/release/last-commit-game
```

## Управление

| Клавиша        | Действие                         |
|----------------|----------------------------------|
| **WASD** / стрелки | Движение                    |
| **E**          | Подобрать предмет (рядом)        |
| **F**          | Поговорить с NPC                 |

## Игровые объекты

- **Синий квадрат** – игрок  
- **Красные квадраты** – баги (отнимают здоровье при касании)  
- **Зелёный квадрат** – печенька (+5 здоровья)  
- **Жёлтый квадрат** – кофе (увеличивает скорость)  
- **Голубой квадрат** – сисадмин (диалог)  
- **Оранжевый квадрат** – тестировщик (диалог)  
- **Серые квадраты** – стены  

## Разработка

Проект построен на [Bevy 0.15](https://bevyengine.org/).  
Архитектура ECS разделена на компоненты (`components.rs`) и системы (`systems.rs`).  
Принимаем пул‑реквесты! Идеи и баги в Issues.
